# Thread

## Create thread

`POST https://rebyte.ai/api/sdk/threads`

Create a new thread.

### List threads

`GET https://rebyte.ai/api/sdk/threads`

Get list of threads.

### **Get thread**

`GET https://rebyte.ai/api/sdk/threads/{thread_id}`

Get a thread by id.


### **Update thread**

`POST https://rebyte.ai/api/sdk/threads/{thread_id}`

Update a thread by id.
