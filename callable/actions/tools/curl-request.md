# Http Request

### Description
Send Http/Https Requests

**Parameters:**
### spec
| Parameter | Description                              |
|-----------|------------------------------------------|
| scheme    | HTTP or HTTPS                            |
| method    | GET or POST                              |
| host      | host of URL                              |
| header    | header of HTTP request                   |
| body      | post body, only available in POST request |

### config
None
