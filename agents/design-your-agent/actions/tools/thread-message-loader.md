# 线程消息加载器

此动作允许您在代理中加载线程的历史消息。

## 使用方法

* 首先在您的代理中添加一个`线程消息加载器`动作。

<figure>
  <img src="../../../../images/thread-1.png" />
</figure>

* 设置"线程上的最近消息"来指定您想要加载的最近消息数量。

<figure>
  <img src="../../../../images/thread-2.png" />
</figure>

* 在后续动作中使用加载的消息来为对话添加上下文。

* 运行代理并查看从线程加载的消息。

<figure>
  <img src="../../../../images/thread-3.png" />
</figure>
